Le projet


Noise-Planet
Outils scientifiques pour l'évaluation du bruit dans l'environnement

Le projet Noise-Planet vise à fournir un cadre global et générique dédié à la collecte, à la modélisation et à l'évaluation du bruit dans l'environnement.

Ce projet est divisé en trois parties principales :

  • L'Infrastructure de Données Spatiales OnoMap (décrite ici),
  • le module de "mesure",
  • le module de "modélisation".



Module de "mesure"

Ce module est dédié à la mesure du bruit. Les données peuvent provenir de différentes sources telles que des applications pour smartphone, des capteurs....
Pour l'instant, ce module est alimenté uniquement par l'application Android NoiseCapture.

Module de "modelisation"

Ce module traite des outils développés et utilisés pour modéliser la propagation du bruit et pour calculer les cartes de bruit.
Un des principaux outils présentés ici est NoiseModelling (préalablement nommé "NoiseM@p"), un plugin pour le SIG OrbisGIS.
